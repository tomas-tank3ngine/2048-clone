
# Play 2048

Play the hit mobile game 2048 in-browser and on mobile! Merge tiles of the same value on a 4x4 grid, combining until you reach 2048... or run out of room!


## Features

- Responsive layout for mobile and desktop
- Swipe on mobile | arrow keys on Desktop


## Tech-Stack
- HTML | CSS | JavaScript
- NextJS
- React 18
- TypeScript
- Jest
## Demo

https://2048clone.netlify.app


## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install lodash
  npm install next
  npm install react
  npm install react-dom
  npm install react-responsive
  npm install uid
```

Start the server

```bash
  npm run start
```
